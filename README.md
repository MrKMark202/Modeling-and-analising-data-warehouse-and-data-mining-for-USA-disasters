# Zavrsni rad
 - Repozitorij za završni rad iz predmeta Skladište i rudarenje podataka.
 - Tema je: Modeliranje i analiza podataka nad podacima o katastrofama u SAD-u.
 - Cilj završnog rada: Obraditi podatke i analizu nad podacima o katastrofama u SAD-u i na kraju predstaviti rezultat analize.
 - Iz rezultata je bio cilj izvuči informacije koje mogu pomoći SAD-u oko bolje procjene i pregleda sveukupne štete koju je napravila određena katastrofa.
